STEP 1: Get Your Resend API Key
========================================

1. Go to: https://resend.com/signup
2. Sign up for FREE account (no credit card needed!)
3. After signup, go to: https://resend.com/api-keys
4. Click "Create API Key"
5. Give it a name (e.g., "CareerHub Dev")
6. Copy the API key (starts with "re_...")

‚ö†Ô∏è Important: Save the API key immediately - you won't see it again!

========================================
STEP 2: Add API Key to .env
========================================

Open your backend/.env file and add:

RESEND_API_KEY=re_your_actual_api_key_here
FRONTEND_URL=http://localhost:5173

Example:
RESEND_API_KEY=re_123abc456def789ghi
FRONTEND_URL=http://localhost:5173

========================================
STEP 3: Restart Backend Server
========================================

Stop your backend server (Ctrl+C) and restart:

cd backend
npm start

========================================
PRODUCTION SETUP (OPTIONAL)
========================================

For production, you should verify your own domain:

1. Go to Resend Dashboard > Domains
2. Click "Add Domain"
3. Enter your domain (e.g., careerhub.com)
4. Add the DNS records shown
5. Wait for verification
6. Update sendEmail.js:
   from: 'CareerHub <noreply@yourdomain.com>'

Without verified domain, emails go from:
from: 'CareerHub <onboarding@resend.dev>'

This works fine for development!
